# SENG-321 Requirements Engineering

I worked on all aspects of the document, including the:

- Introduction
- Overall Description
- System Features
- External Interface Requirements
- Other Non-Functional Requirements
- Other Requirements
- Use Cases
- Data Flow Diagrams
- User Interface
- Appendix
